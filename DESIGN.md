# DESIGN

## Dirección visual
Dashboard móvil primero, sobrio y práctico. Sensación de cuaderno de entreno moderno: oscuro, legible, con acciones claras.

## Jerarquía
1. Sesión activa y timer.
2. Botones para Día A / Día B.
3. Tabla de ejercicios rellenable.
4. Recomendación semáforo.
5. Estadísticas simples.

## Sistema visual
- Fondo oscuro `#0b1120` con paneles `#111827`.
- Verde/ámbar/rojo para semáforo de esfuerzo.
- Botones grandes táctiles.
- Tipografía sistema para velocidad y legibilidad.
- Pocos bordes, alto contraste, inputs grandes.

## Anti-patrones
- Gráficas decorativas sin datos suficientes.
- Formularios largos antes de empezar.
- Cards anidadas innecesarias.
- Lenguaje técnico tipo RPE 7.5 sin traducción práctica.
